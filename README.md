# Tailwind Color Picker

Tailwind Color Picker is a VS Code extension that provides:

✔ Inline color preview for Tailwind CSS classes  
✔ Color picker command for Tailwind palette and arbitrary colors  
✔ Support for Tailwind v3 JIT arbitrary values  
✔ Support for `@apply` (including nested selectors and @layer)  
✔ Support for class attributes in HTML, React, Vue, Svelte  
✔ Support for `theme()` function  
✔ Support for `--tw-ring-color` and `--tw-ring-offset-color`

---
